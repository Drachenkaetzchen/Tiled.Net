# Tiled.Net
A .NET library for reading Tiled maps.

## About Tiled.Net
Tiled.Net provides a general way of reading and writing TMX (Tiled Map XML) files. Tiled.Net aims to be usable everywhere, rather than being tied to things like XNA/MonoGame, Unity, etc. 

## Dependencies
Tiled.Net uses .NET 4.5 and requires no external dependencies.
